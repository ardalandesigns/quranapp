Endpoint Strategy: To get both the exact Uthmani text (minimal) and the Kurdish Sorani translation in a single request, we use the `alquran.cloud` API which hosts Tanzil data.

API Request Details

Base URL

http://api.alquran.cloud/v1/

Endpoint (Get Surah Al-Fatihah)

surah/1/editions/quran-uthmani-min,ku.asan

Parameters:
`1`: Surah Number
`quran-uthmani-min`: Exact Tanzil text without extra tajweed marks.
`ku.asan`: Kurdish Sorani Translation (Tafsir Asan).

Fetching Example (JavaScript)

async function fetchSurah(surahNumber) {
    const url = `https://api.alquran.cloud/v1/surah/${surahNumber}/editions/quran-uthmani-min,ku.asan`;
    try {
        const res = await fetch(url);
        const data = await res.json();
        return data;
    } catch (err) {
        console.error("Error fetching Surah", err);
    }
}
